Handy Chrome extension

Web contents as a Google Task

Save as a Google Task:

  • a web page title together with the page address
  • choose any text block as the task's title or details

Privacy policy

The extension does not collect any user data. The extension only mediates a convenient transfer of a web page contents from a user's browser to the user's Google Tasks tool. The extension does not transmit any data anywhere except the Google Tasks tool.

For its function, the extension requires and requests only one Google OAuth 2.0 scope: https://www.googleapis.com/auth/tasks.

If granted by a user, this scope enable the extension to create, edit, organize, and delete all user's tasks.

The access token obtained after a user's authentication with the Google Identity is used only for accessing the Google Tasks API.

Of the possible Google Tasks API operations, the extension performs two:

  • List Tasklists (GET https://tasks.googleapis.com/tasks/v1/users/@me/lists) returning descriptions of the user's task lists.

  • Insert Tasks (POST https://tasks.googleapis.com/tasks/v1/lists/{tasklist}/tasks) creating a new task in the specified task list.

The list of a user's tasklists is transformed into options of a SELECT HTML element that is needed to allow the user to select a target tasklist into which a new task is to be saved. The tasklist selected in the SELECT element is saved into user's browser localstorage so that the previously selected tasklist can be automatically selected when the user reuses the extension.
